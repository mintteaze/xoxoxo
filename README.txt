Порядок запуска программы
* Запускаем командную строку и вводим следующие команды

  ```
  cd {имя папки} # и так продолжаем пока не дойдем до папки с проектом
  ```

* Затем вводим данную команду (для установки нужных для работы модулей)

  ```
  pip install -r requirements.txt
  ```

* Запускаем файл main.py и переходим по данному адресу:
  ```
  http://127.0.0.1:5000
